# 🎮 SteamAgentRecommendator

> Sistema inteligente de recomendación de videojuegos RPG dentro de la plataforma Steam

---

## 📌 Descripción

**SteamAgentRecommendator** es un agente inteligente diseñado para recomendar videojuegos de tipo **RPG** dentro de la plataforma **Steam**, una de las mayores tiendas de distribución digital de videojuegos del mundo.

Steam ofrece un catálogo dinámico con miles de títulos, donde los usuarios pueden:
- 🛒 Comprar juegos  
- ⬇️ Descargarlos  
- ⭐ Valorarlos  
- ⏱️ Registrar su tiempo de juego  

---

## 🧠 Enfoque del Proyecto

A diferencia de sistemas tradicionales, Steam utiliza un modelo de clasificación **híbrido**, lo que hace más complejo el proceso de recomendación:

### 🎯 Sistema de Clasificación
- **Categorías oficiales** (géneros principales)
- **Tags descriptivos** (mecánicas, ambientación, estilo)
- **Subgéneros emergentes** (combinaciones de tags)

### 🏷️ Origen de las etiquetas
- 👨‍💻 **Desarrolladores** → asignación inicial  
- 👥 **Comunidad** → votación y ajuste dinámico  

---

## ⚙️ Problema a Resolver

El objetivo principal es **modelar las preferencias del usuario** para recomendar juegos RPG relevantes en un entorno caracterizado por:

- 🔄 **Dinamicidad** → nuevos juegos, actualizaciones y descuentos constantes  
- 👀 **Observabilidad parcial** → las preferencias se infieren del comportamiento  
- 📈 **Evolución** → los gustos cambian con el tiempo  
- 📊 **Alta dimensionalidad** → gran volumen de datos implícitos y explícitos  

---

## 🚀 Objetivo del Sistema

Desarrollar un agente inteligente capaz de:
- Interpretar tags y relaciones entre juegos  
- Adaptarse a cambios en preferencias  
- Generar recomendaciones personalizadas de alta calidad
